# Obfuscator · public issue tracker

This is the public tracker for Obfuscator, a Code Stage asset on the Unity Asset Store.

Use it to report a bug (something broken or behaving unexpectedly) or request a feature (a capability you'd like to see). Open a new issue with the buttons above, or use the in-editor "Report a Bug / Request a Feature" panel inside the asset. Every report gets a status label, so you can see where it stands from triage through to a shipped fix.

This repo tracks reports and requests only. It doesn't contain the asset's source code. For license or purchase questions, email support instead.
